## Descripción General

Este repositorio contiene la solución completa del Taller Integrador de la Sesión 14 de la asignatura Pruebas de Software.

El proyecto aborda tres áreas fundamentales del aseguramiento de calidad y optimización de software:

- Testing Dinámico con cobertura total de ramas.
- Property-Based Testing utilizando Hypothesis.
- Optimización de selección de casos de prueba mediante Programación Dinámica (Knapsack 0/1).

---

## Objetivos Alcanzados

- Implementación de pruebas unitarias exhaustivas con Pytest.
- Cobertura del 100% de ramas en el módulo analítico.
- Validación automática de propiedades matemáticas mediante Hypothesis.
- Comparación entre algoritmos Greedy y Programación Dinámica.
- Demostración empírica de la superioridad de la solución óptima.
